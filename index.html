1. Tujuan

Setelah mempelajari modul ini, peserta didik diharapkan dapat:

  • Memahami dasar pemrograman web menggunakan HTML (HyperText Markup Language).
  • Mengenal berbagai tag dan elemen dasar HTML.
  • Menyusun halaman web sederhana menggunakan HTML.
  • Menyelesaikan beberapa latihan pemrograman dasar HTML secara mandiri.

2. Dasar Teori

2.1 Pengertian HTML

HTML (HyperText Markup Language) merupakan bahasa markup untuk menyusun halaman web yang dapat dibaca oleh browser. Browser seperti Google Chrome atau Firefox akan menerjemahkan kode HTML menjadi tampilan visual halaman web.

2.2 Beberapa Elemen HTML yang Sering Digunakan

Tag Fungsi
<h1> - <h6> Judul atau heading
<p> Paragraf teks
<a> Hyperlink
<img> Menampilkan gambar
<br> Ganti baris
<hr> Garis pemisah horizontal
<b> Teks tebal
<i> Teks miring
